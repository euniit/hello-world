# hello git

## git 명령어 요약

- clone: 원격 저장소를 내 컴퓨터에 복사해 온다.
- add: 내 컴퓨터에서 작업한 파일들을 스테이지에 추가한다.
- commit: 스테이지에 올라온 파일들을 가지고 내 컴퓨터에 저장한다.
- push: 커밋들을 원격 저장소에 업로드한다.

## 브랜치 변경하기
- 브랜치: 기존 내용을 유지한 채 새로운 내용을 추가하고 싶을 때 사용한다.
- 체크아웃: 특정 브랜치(혹은 커밋)으로 돌아가고 싶을 때 사용한다.

## 충돌 해결하기
- 제일 중요한 점: 겁내지 말아요!
